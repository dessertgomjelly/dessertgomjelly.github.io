# 쉘 스크립트 란?

-  Shell Script는 유닉스 또는 리눅스 시스템에서 명령어를 자동화하는 데 사용 되는 텍스트 파일이다.
-  여러 개의 쉘 명령어와 스크립트 언어로 작성된 명령어들이 순차적으로 실행되도록 작성된 스크립트이다.

```
 터미널에 입력했던 여러 명령어들을 하나의 스크립트 파일에 나열하여, 스크립트 파일이 자동으로 실행 함으로 효율적이면서 간편하게 작업 처리를 할 수 있는 것이다.
```
